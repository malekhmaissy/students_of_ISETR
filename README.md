# ISETR Admissions Analytics Dashboard

> **Interactive Power BI dashboard providing deep insights into admissions trends, student performance, and enrollment patterns across engineering programs at ISETR (2022-2024)**



## Overview

This Power BI project delivers comprehensive analytics for the **Institut Supérieur des Études Technologiques de Radès**, enabling data-driven decision-making around student admissions, academic performance, and program-level enrollment strategies.

The dashboard transforms raw admissions data into actionable insights, making it easy for administrators, academic coordinators, and institutional stakeholders to understand admission trends, identify patterns, and benchmark performance across five distinct engineering specializations.


## Key Features

### **Real-Time KPI Monitoring**
- **Performance Grades Distribution**: Track student performance across four tiers (Passable, Assez Bien, Bien, Très Bien)
- **Admission Status Breakdown**: Visual pie chart showing admitted vs. repeated student ratios
- **Dynamic Metrics**: Real-time percentage changes for conditional admissions, accepted students, and repeating cohorts

### **Multi-Dimensional Analysis**
- **Program Comparison**: Side-by-side analysis across 5 engineering specializations
  - GC (Génie Civil)
  - GE (Génie Électrique)
  - GM (Génie Mécanique)
  - SEG (Systèmes Électroniques et Gestion)
  - TI (Technologie de l'Information)

- **Temporal Insights**: Year-over-year comparisons (2022, 2023, 2024) with animated transitions
- **Departmental Breakdown**: Enrollment volumes segmented by primary and alternate program tracks

### **Interactive Filtering**
- Program-level drill-down capabilities
- Year-wise data slicing with smooth animations
- Global level selector for comprehensive or granular viewing


## Dashboard Components

| Component | Purpose |
|-----------|---------|
| **KPI Cards** | Display admission performance distribution percentages |
| **Pie Chart** | Show admission acceptance rate vs. repetition rate |
| **Grouped Bar Chart** | Compare admission volumes across departments and tracks |

## Getting Started

### Prerequisites
- Microsoft Power BI Desktop or Power BI Service access
- Data source connectivity (SQL Server, Excel, or cloud-based data warehouse)

### How to Use
1. **Filter by Program**: Select from GC, GE, GM, SEG, or TI to focus on specific specializations
2. **Choose Academic Year**: Use the year selector (2022, 2023, 2024) to view historical trends
3. **Explore Performance Metrics**: Hover over KPI cards to see detailed breakdowns
4. **Analyze Trends**: Play the animation to visualize enrollment progression over time
5. **Download Insights**: Export visualizations for presentations and reports


## Key Insights at a Glance

- **38%** of students achieved "Assez Bien" (Good) performance level
- **78.54%** of admitted students successfully completed their programs
- **SEG Program** leads with **2,359** admissions, indicating high institutional demand
- **Year-over-Year Growth**: Admissions increased **77.70%** across all programs
- **Consistency**: Repeat rate stabilized at **21.23%**, showing predictable enrollment patterns
